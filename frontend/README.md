# Portfolio Dashboard (Frontend)

Next.js 14 dashboard that displays portfolio metrics and asset tables. Consumes the Portfolio API backend.

## Setup

Set `NEXT_PUBLIC_API_URL` to the backend API base URL (e.g. `http://localhost:8000` for local).

```bash
npm install
npm run dev
```

Open http://localhost:3000.

## Scripts

- `npm run dev` — Development server
- `npm run build` — Production build
- `npm run start` — Run production server (after build)
- `npm run lint` — Run ESLint
